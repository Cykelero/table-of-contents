Installation

You can turn on Table of contents in Safari Settings:

  • Open the Settings app.
  • Navigate to Safari, then Extensions.
  • Find Table of contents, and switch it on.

Usage

Once the extension is enabled, you can invoke it from Safari's toolbar menu.

If you have a keyboard connected, you can also trigger the list using ⌃G.

You can turn on Table of contents in Safari Extensions preferences.

Once enabled, show the list using the toolbar button, or by pressing ⌃G.
